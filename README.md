# Earth View

Standalone web app for the 3D Earth View visualizations from Life Manager.

## Development

```sh
npm install
npm run dev
```

Open `http://localhost:5175`.

## Build

```sh
npm run build
```

The deployable static site is emitted to `dist/`.

## Deploy

This is a Vite static app. Deploy `dist/` to Cloudflare Pages, Netlify, Vercel, or any static host.
